# 太上老君说常清静经

## 概述

《清静经》全称《太上老君说常清静经》，唐代道教经典，传为太上老君（老子神格化）所说。是道教内丹和心性修炼的核心经文之一，篇幅短小精悍（约400字），以"清静"为总纲，论述了从遣欲澄心到悟空得道的完整修道路径。

## 核心结构

经文分三大段，层层推进：

### 第一段：道体论
"大道无形，生育天地；大道无情，运行日月；大道无名，长养万物。"
- 确立[[道]]的超越性：无形、无情、无名
- 道虽超越，但作用是实在的——生天地、运日月、养万物

### 第二段：修道路径
从"夫道者，有清有浊"到"能悟之者，可传圣道"。
- 清浊动静的辩证关系
- 遣欲 → 澄心 → 悟空 → 真静的进阶
- "空"的层层破除：观空 → 空无所空 → 无无亦无 → 湛然常寂

### 第三段：障碍与检验
从"上士无争"到结尾。
- 上士与下士的区别：争 vs 不争
- 妄心 → 烦恼整条因果链
- 最终回归"常清静矣"

## 核心命题

### 人能常清静，天地悉皆归
这是全经的核心主张。[[清静]]不是消极的什么都不做——是回到本源之后，天地万物自然归位。不是你去掌控天地，是你清静了，天地自己就对了。

### 遣其欲而心自静，澄其心而神自清
修行的下手处：欲望和杂念。不是直接追求"静"，是通过遣欲来达成静。方法论很具体——从"做减法"入手。

### 虽名得道，实无所得
反高潮的设计。"得道"是个方便说法。真得了的人知道自己什么都没得——因为道本来就在，从来没丢过。说"得"是为了"化众生"。

### 妄心 → 永失真道
完整的堕落链条：妄心 → 惊神 → 著万物 → 贪求 → 烦恼 → 流浪生死 → 永失真道。起点是"妄心"——那颗不安分的心。

## 实践价值

对赵总而言，清静经不是玄学读物——是实操手册：
- **遣欲**：赵总说过"不走极端守中"，清静经给了方法论——从减欲望开始，不是强行压制，是让它自己生不起来
- **常应常静**：不是逃避生活，是在生活中保持清静。该管钱管钱，该带娃带娃，心里不乱
- **虽名得道实无所得**：赵总对道的态度——"懂我自己比进入一更重要"，与经文的"为化众生名为得道"异曲同工。不执着于"得道"这个概念本身，就是真修行
